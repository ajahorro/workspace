🚀 HOW TO RUN
================================================================================

Step 1: Install Dependencies
$ npm install

Step 2: Set Up Supabase (3 minutes)
1. Create project at https://supabase.com/dashboard
2. SQL Editor → Copy DATABASE_SCHEMA.sql → Run
3. Storage → Create buckets: receipts, chat_media (both public)
4. Database → Replication → Enable realtime for:
   - bookings
   - booking_vehicles
   - payments
   - notifications
   - booking_messages
5. Settings → API → Copy credentials

Step 3: Configure Environment
$ cp .env.example .env
(Edit .env with your Supabase credentials)

Step 4: Run
$ npm run dev

Open http://localhost:5173

Step 5: Create Super Admin
1. Register through the app
2. In Supabase SQL Editor:
   UPDATE profiles SET role = 'SUPER_ADMIN' WHERE email = 'your-email@example.com';
3. Log out and log back in
